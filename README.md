# Learn React + TypeScript from Portfolio

This project is for learning React from the basics using Asabeneh's **30 Days Of React** as the backbone, but adapted to a modern React + TypeScript + Vite workflow and connected back to the actual portfolio project.

Source backbone:
https://github.com/Asabeneh/30-Days-Of-React

Portfolio GitHub:
https://github.com/sudharsan-ak/personal-portfolio/

Live portfolio:
https://sudharsansrinivasan.com/

## Purpose

The goal is to build React muscle memory in VS Code and explain the portfolio clearly in interviews.

The learning path should focus on:

```text
React
TypeScript
Vite
Function components
Hooks
Modern React patterns
Portfolio-style exercises
Interview-ready explanations
```

## Source files

```text
current_status.md        -> current progress and next step
learning_roadmap.md      -> 30-day plan and local app structure reference
project_rules.md         -> teaching, review, notes, upload, and workflow rules
react_rules.md           -> cumulative React + TypeScript coding rules
mistakes_log.md          -> recurring mistakes, prompt mismatches, and gotchas
portfolio_mapping.md     -> verified mappings to the real portfolio repo/live site
interview_wording.md     -> human-ready interview explanations
learning_index_part1.md  -> Week 1 historical progress archive
learning_index.md        -> deprecated stub only, if still present
```

Daily and weekly notes are separate study files:

```text
dayX_notes.md
weekX_notes.md
```

## Before starting a new day

Start with:

```text
current_status.md
project_rules.md
learning_roadmap.md
portfolio_mapping.md
interview_wording.md
learning_index_partX.md
latest day/week notes
```

Use these as reference when relevant:

```text
react_rules.md
mistakes_log.md
```

`react_rules.md` and `mistakes_log.md` are not meant to be fully reread every time. They are cumulative reference files for reviews and topic-specific reminders.
